## 行动缓冲区介绍

由于大招/追架攻击等特殊插队技能的存在, 只有正常的行动条是不可行的, 需另有一个不按照 AV 值排序, 按照入队顺序 FIFO
的行动缓冲区.

这个功能需要修改当前 ECS 系统的架构(主要是行动条系统)

### 与行动条系统的交互

当一个对象即将行动, 则把它从行动条顶(AV=0)弹出, 压入缓冲队列.
如果缓冲队列目前为空, 行为上就与没有缓冲区一样

### 与角色大招的交互

角色大招默认插入位置为缓冲区最后一个角色大招后. 即:

1. 如果缓冲里只有一个正常回合/追击回合:
   BUFFER: [ent1, ent2 ...] 这里面没有一个是 ultimate/终结技 的行动
   释放终结技后,队列为
   BUFFER: [ultimate1, ent1 ...]
2. 如果已有终结技行动
   BUFFER: [ultimate1, ent1, ent2 ...] 这里面没有一个是 ultimate/终结技 的行动
   释放终结技后,队列为
   BUFFER: [ultimate1, ultimate2, ent1 ...]

## 角色部分设计

```text
角色1
[技能叠层]       [能量/max]¹
HP============--------------
  ===============
[buff]              [HP/Max]
```

这里, `角色1` 是角色名字, 能量/max 对应当前能量与终结技能量, `¹` 这类是按键提示;

HP条分为两层.

- 上层: `======` 是当前的, 用天蓝色配色, `-------` 是已经被消耗的, 用灰色;
- 下层同理, 但下层展示盾量. `====` 用白色(默认颜色), `-----` 不显示

### 盾条计算方法

假设当前HP上限是 max 点, 当前盾量是 x 点. 那么, 盾条长度为:

width * (x / max)

width 是盾条(血条)渲染最大宽度. 如果 x > max, 不溢出显示, 顶满盾条即可.

## 敌方设计

```text
[BOSS]                                                    [弱点]
================================================================
=================================--------------------------[50%]
[buff]                                                [特殊叠层]
```

整个组件居中. 如果有配置再显示, 默认不显示.

`[BOSS]` 是名称, 第一个 `=======` 条是韧性条,默认是满的,会随战斗进行而被削减. 使用白色配色; 禁用状态下(不可被削减韧性时)用灰色(dimmed)
第二个条是血条, `=====` 用红色. 当敌方血量被削减时, 做一个缓冲

- 对应被削减数量的血条先变为白色, 再变为 `-----` . 给用户一个明确的提示

buff部分用于显示boss所具有的一些buff.

`[弱点]` 部分用于显示敌方具有的弱点. 当弱点被禁用的时候, 所有弱点图标改灰色(dimmed)

弱点列举如下:

```python
    PHYSICAL = "physical"  # 物理 D1D2D1
    FIRE = "fire"  # 火 F46953
    ICE = "ice"  # 冰 83BCDD
    LIGHTNING = "lightning"  # 雷  D37FE2
    WIND = "wind"  # 风  63CF9A
    QUANTUM = "quantum"  # 量子  5952C3
    IMAGINARY = "imaginary"  # 虚数   F5E253
```

渲染原则:
以 `物火冰雷风量虚` 中一个字来代表对应弱点, 并使其显示对应颜色即可.

## 选择器设计

选择器只有一个(对我方角色和敌方目标是同一个, 只不过选择时箭头方向不一样罢了)

### 一些不同的选择类型

需要注意的是, 它不只有以下几种, 还有可能自定义其他组合方式. 为了灵活起见, 应该设计一个接口, 调用方可以自己传入规则

如 传入一个主目标, 四个副目标这种选择器类型.

#### 扩散

```text
 △                        ▲                         △
```

这个是扩散类型的伤害选择, 中间为主目标, 其余为副目标. 主目标用 实心 , 副目标用 空心

#### 单体

```text
▲
```

#### 群攻

```text
 △                        △                         △                 △               △
```

群攻/群体目标的选择器统一使用空心. 每一个目标(如果对敌方那就是所有敌方目标, 如果技能对我方释放, 如群体奶, 那就是所有我方目标)

### 着色

对我方, 使用和我方 HP 条一样的颜色即可(天蓝色)
对敌方使用红色

### 朝向

对我方使用下三角

### 对齐

自动计算
